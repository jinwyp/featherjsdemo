为什么要做这个项目, 请看用户在weibo反馈的问题

而且目前没有的促销功能:

1 商品数量限制购买

2 时间限制购买 并显示

3 购买指定数量的有优惠

4 券的功能

 

了解促销现有形式:

 

目前的促销形式:  抢购   渠道促销  满减  组合购  换购  赠品     其中 满减 换购 赠品 是全场类型的促销.  而抢购是单品促销

该项目于新版首页和分类页面有些关联 包括推荐位部分
