单品抢购  在商品详情页面 各种状态解析:

1.1 普通状态 显示 会员价

1.2 设置了一段时间的抢购价格, 并且限制了一段时间库存销售数量. 促销时间到了后会自动变回会员价

1.3 在1.2的基础上 增加限制同一个用户的购买数量.

注意 如果限制了用户购买数量例如每个用户限购两个. 那么 用户在买2个的时候可以用促销价格购买, 如果用户购买超过2个,那么超出的商品数量将会使用会员价格购买. 同时购物车页面 会分开显示促销价格和会员价格的商品

1.4  设置了同一个商品购买多件打折  促销信息 要显示 "满2件每件立减20!  满3件每件立减60!  满6件每件立减80! "
