1 内容模型 Content Type :所有的内容形式都可以理解为内容模型.例如:文章是一种内容模型,相册,视频也是一种内容模型.车系也是一种内容模型, 品牌也是一种内容模型. 会员信息也是一个种内容模型.综合来说,所有要显示的同一种内容类型都可以定义为内容模型Content Type.
2 属性(字段)fields:在一个内容模型里，包含若干个属性(字段)fields 来表述该内容模型, 例如 文章的内容模型Content Type 就有 标题,短标题,导语,正文,等属性fields.
