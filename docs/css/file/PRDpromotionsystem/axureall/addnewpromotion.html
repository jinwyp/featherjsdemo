排他条件主要是两种: 排除 单品促销  和 满减券促销

 

1 无排除条件 即 一个商品可以同时使用 单品促销的价格,而且在购物车结算页面 还可以同时使用全场的促销.

例如 一个商品的会员价为100元, 那么设置单品减价促销 的价格 为90元, 那么 在购物车时, 同时用90元价格计算,同时还可以使用 满200减30元的 全场促销.

 

2  关于排除单品促销, 和刘丹确认了一下, 实际上这个需求不多.  即 如果有两个单个商品 同时减价促销.

例如  商品A  会员价为 100元  促销价为80元  商品B 会员价为150元  促销价格为120元. 

如果设置了商品A的排除单品促销 商品B, 那么以80元购买商品A后,就不能在用120元购买商品B.  

但这个需求不多.

 

 

所以 单个商品的 排除单品促销需求是不需要的. 不需要做.

 

同理 全场满减促销的设置中, 所有全场满减的促销默认都是互斥的.只需要排除单品促销即可. 

全场满减排除单品 不需要在全场满减促销中设置,  而是统一在单品促销中设置 排除全场满减促销.

 

全场满减券 和全场换购  还可以选择 同时和所有全场设置同时使用.
