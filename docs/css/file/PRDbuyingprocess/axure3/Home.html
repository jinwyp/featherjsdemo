送货时间修改逻辑

 

1 不用新增字段  但

 

当选择了母婴之家快递后 出现 选择 送货时间 : 只工作日送货 / 工作日双休日与假日 送货 / 只双休日送货

 

如果选择的是 其他快递 则不显示 该 送货时间部分.

 

 

2 当选择了母婴之家快递后 出现 选择 送货时间 :

2.1  只工作日送货  送货时间  如果今天是工作日(即周一到周五)那么今天下单, 送货时间为明天的00 零点时间.  如果今天不是工作日(即周六或周日) 那么今天下单,送货时间为下周一的零点时间

 

2.2 工作日双休日与假日 送货   改为 今天下单, 送货时间为明天的00 零点时间.

 

2.3 只双休日送货  如果今天是双休日(即周六或周日) 改为 今天下单, 送货时间为明天的零点时间. 如果今天不是双休日 则 为送货时间 为 本周 星期六的00 零点时间.

同时 "节假日 文字"先删除了.
